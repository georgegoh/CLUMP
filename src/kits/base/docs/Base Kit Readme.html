Project Kusu : Base Kit Readme
This page last changed on Nov 08, 2007 by najib.

This kit provides the following software:

  • Kusu Runtime
    • Kusu Runtime includes tools like addhost, ngedit, repoman, repopatch, the Kusu python libraries for cluster management, operation and deployment and initrd images needed for imaged and diskless nodes.
  • Modules
    • The Environment Modules package provides for the dynamic modification of a user's environment via modulefiles.
  • Pdsh
    • Pdsh is a high-performance, parallel remote shell utility.

This kit provides the following components:

  • component-base-installer
    • This component provides the node with the Kusu management tools for
      installer-type nodes. This component is associated with the Installer nodegroup by default.
  • component-base-node
    • This component provides the nodes with the Kusu Runtime for non-installer-type nodes. This component is associated with the compute, compute-imaged and compute-diskless nodegroups by default.
