Install this kit

Use the 

kitops
command to install a kit.

  1. Import the kit information to the database, and copy files from the media. To do this, use the kitops command: 

    kitops -a -m path_to_kit

    where path_to_kit can be an ISO, URL, FTTP path, or NFS path.

    Note:

    Note the definitions of these command options:

    • -a: Adds a new kit.

    • -m: Points to the ISO (media) containing the kit files.

    The kit and its components are registered in the database, and the kitâ€™s contents are copied to this directory on the primary install node:

    /depot/kits/kit_name/version/architecture

  2. Add the kit to a repository on the primary install node. To do this, use the repoman command:
    1. Determine which repository on the primary install node to which you want to add the kit. To do this, list the repositories currently in the system. 

      repoman -l

    2. Add the kit to the desired repository. 

      repoman -r "repository name"
      -a -k kit_name

    3. Refresh the repository. 

      repoman -r "repository name" -u

    Note:

    Note the definitions of these command options:

    • -l: LIsts the repositories currently in the system.

    • -r: Selects the repository by name.

    • -a: Adds the kit to the repository.

    • -k: Points to kit you are adding to the repository.

    • -u: Rebuilds the repository metadata with the newly added kits.

  3. Associate kit components with nodegroups. To do this, run ngedit from a command prompt to launch the Node Group Editor tool.
    1. Within the Node Group Editor, navigate to the Repository screen.
    2. Ensure the nodegroup is using the repository that contains the new kit, and then select Next.
    3. Navigate to the Components screen.
    4. Select the component(s) you want to install on all nodes in the nodegroup.
    5. Save any changes you have made.
  4. From the Node Group Editor, you can update existing nodes in the nodegroup with the new kit components. The last screen in ngedit asks if you want to push the kit package to the nodes using the cfm tool.

    You can also choose to run cfm manually at a later time, or simply reboot the nodes to force a reinstall of the new packages.

    Note:

    Any new nodes added to the nodegroup after saving your changes within ngedit automatically have the kit package installed.
